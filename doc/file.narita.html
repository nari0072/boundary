成田の研究目的

粒子の境界のエネルギーを第一原理計算ソフトVASPで求める．

岩佐の失敗の理由

  1. 作られたモデルを第一原理計算ソフトVASPを用いて 構造緩和を行い系全体のエネルギーを計算
  2. エネルギーは予測通り，下がった結果が得られたが，構造緩和に過ちがあった．
  3. 気が付かなかった理由は，結果として図にしなかったから．

成田の研究目的

  1. 「人は楽なことしかしない」から，成田は視覚化を楽にするソフトを開発する．
  2. 手法Cairo Rubyで2次元の視覚化を容易に実現するCairoを用いて粒界原子の構造を表示するソフトを開発します．Cairoは多様な出力デバイスをサポートするための2次元のライブラリーです．特にSVGの png, jpgは重たくて，画質が悪い．SVG(scalable vector graphics)はスケール化できるベクトル表示が可能．
